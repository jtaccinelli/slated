# Slated — Monorepo Root

This repository is a monorepo containing three subprojects. Each subproject has its own `CLAUDE.md` with project-specific instructions.

---

## Subprojects

| Directory  | Purpose                                                                                                     |
| ---------- | ----------------------------------------------------------------------------------------------------------- |
| `slated/`  | The Slated framework itself — agentic coordination tools, roles, backgrounds, skills, and agent definitions |
| `landing/` | Marketing landing page for the Slated framework                                                             |
| `docs/`    | Documentation site for the Slated framework                                                                 |

---

## Working in This Repo

When working on a specific subproject, navigate to its directory and follow the instructions in its `CLAUDE.md`. Do not apply patterns or conventions from one subproject to another without verifying they are compatible.

The `slated/` subproject uses the Slated agentic coordination framework — read `slated/CLAUDE.md` before doing any work there.

The `landing/` and `docs/` subprojects are standard web projects and do not use the Slated framework for their own development.
